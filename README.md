# SIMPOS
SIMPOS (Sistem Informasi Manajemen Posyandu) adalah aplikasi berbasis Java NetBeans dan MySQL yang digunakan untuk membantu kader dalam mengelola data balita, ibu hamil, imunisasi, pemeriksaan kesehatan, jadwal posyandu, serta penyampaian informasi kepada masyarakat. Sistem ini membuat pelayanan posyandu lebih cepat, terstruktur, dan efisien.
